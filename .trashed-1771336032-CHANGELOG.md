# Changelog

## [1.0.1] - 2026-01-18
### Added
- `find_dll()` function for flexible RandomX DLL search across multiple folders (`build/`, `windows/`, `tevador/`, `assets/`, root).
- Rolling hashrate statistics (`hashes_since_last`, `last_stat_time`) to display H/s at regular intervals.
- `destroy()` method in `RandomXVM` for proper cleanup of cache, dataset, and VM resources.

### Changed
- Banner and agent string updated to `py-rig/1.0.1`.
- Nonce injection logic improved with `nonce_offset` and `bytearray(blob_template)` for robustness.
- Argument parser refined with clearer defaults and help text.

### Fixed
- Stronger error handling in `stratum_send`, `stratum_recv`, and `shutdown_miner`.
- More informative error messages when DLL is not found.

---

## [1.0.0] - 2026-01-10
### Added
- Initial miner release with pool login, mining loop, submit worker, and basic stats.
- Basic `RandomXVM` class for cache/dataset management.
- Argument parser with options: `--url`, `--user`, `--password`, `--tls`, `--tls-insecure`, `--mode`, `--threads`, `--debug`.

### Changed
- Initial banner displayed `py-rig/1.0.0`.

### Fixed
- Basic shutdown miner implementation and minimal error handling.
